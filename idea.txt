MLoated Juice
Simpen data di cookie, Image disimpen di server bukan blob DB. Trainee boleh pake Jquery, Tailwind, DaisyUI tapi CDN. Semua progress ditulis di progress.txt . pengerjaan juga ada penggunaan AJAX untuk infinite Scrolling pada menu.

Pengerjaan menggunakan architectural pattern MVC. Trainee juga diminta untuk menggunakan best practice Password Hashing, CSRF Prevention, XSS Prevention, Disable function berbahaya php di php.ini (eval, assert, dll). Trainee juga diminta untuk membuat middleware (Logged in user ga bisa ke login page). Diminta juga untuk mengurangi resiko directory listing dari .htaccess nya.
